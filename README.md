# PhytoChem-Tracker
this represents an AI-built and designed phytochemical intelligence platform for rapid retrieval of phytochemical data
PhytoChem-Tracker (PCT) was assembled and built utilizing the AI tools of Google Gemini 3.0 and Google Antigravity for satisfaction of the final project requirements of CS5047
There are still bugs to be worked out, so this is version 1.0 of PCT
